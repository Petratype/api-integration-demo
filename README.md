# PokeAPI Integration Demo

Simple demo that fetches data from [PokeAPI](https://pokeapi.co) and displays basic info about Pikachu.

**Usage:**
- Open `index.html` in your browser  
- Click “Load Pokémon” button  
- Info about the Pokémon will appear below (name, height, weight)

**Tech used:**
- HTML  
- JavaScript (Fetch API)
